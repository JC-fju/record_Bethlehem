# project_Bethlehem

先學會C再來搞這個
總之玩法會想要偏向中出式(mai那種)
有四條判定線 線中間有判定點
有點像是菱形那種感覺
操作方式會像是從正中間跑圖形出來(不一定啦 可能單線或雙線圖形 like:/  \或ˊ ˋ)
操作菱形的點或是線來符合圖案
可能可以加個換位 像是菱形右邊那點移到正下方
大概先這樣吧

喔對了 遊戲的方式之所以這樣是為了有模擬星艦移動的感覺(like cytusll的掃瞄)
滑軌是航線 單判定點是星球 換位是蟲洞或亞空間跳躍等等


背景故事感覺還是cyberpunk一點會比較好操作，然後結合一點神話價值觀吧，最好是那種神愛世人與以人為本的價值觀碰撞。Bethlehem(伯利恆之星)，嘛，畢竟是耶穌誕生時的天體，象徵指引或希望等等，但最合理應該是超新星爆炸或啥的，如果那顆恆星存在的天體系統包含智慧生命的話，嗯，就是滅絕呢(雖然智慧生命應該會想盡辦法逃離就是了)，總之主線可能就會有點老派，為了尋求人類延續，帶著由A.I.精挑細選後的人類搭著去往宇宙的艦船(可能取叫S.A.I.N.T 全名Sanctuary Ark for ideal Nova Tempora 理想新時代的庇護方舟)，巴拉巴拉(先這樣改天繼續)
